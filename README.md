
!!! Free Mac Gyver !!!

This is a game built with python and pygame, which is part of the Python Developer Path from Openclassrooms.
https://openclassrooms.com/fr/paths/68-developpeur-dapplication-python

How to run the program on your computer:

    make sure you have python installed on your computer, or do it at https://www.python.org/downloads/
    install and activate a virtual environment
    download pygame https://www.python.org/downloads/ and install it via pip : python 3 -m pip install pygame
    clone the github repository and install it on your computer with git, or download the .zip version from
    https://github.com/donkansigui/OCR_P3.git
    launch gui.py


------------------------------------------------------------------------------------------------------------------------------------------------------------
                                            !!! RULES !!!



    - press the F1 key to start the adventure, escape to leave the game (once locked in the Maze you will
      not be able to leave the program until you face Murdock!)

    - use the arrows left, right, up, down to move our hero.

    - move our hero on an object to pick it up (collected items appear in the inventory at the bottom of
      the game window).

    - once in possession of the three objects move Mc Gyver on guard to lull the guard and escape from
      the Maze.
